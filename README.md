# Discord Bot

### Necessary permissions
* Scopes
    * bot
    * applications.commands

* Bot Permissions
    * General Permissions
        * Read Messages/View Channels
    * Text Permissions
        * Send Messages
        * Manage Messages
        * Embed Links
        * Attach Files
        * Read Message History
        * Mention Everyone
        * Use External Stickers
        * Add Reactions

### Python version
Python 3.11.2

### Packages to install
* discord
* jproperties
* jproperties
* youtube-dl

Command to install: `pip3 install discord jproperties jproperties youtube-dl`
