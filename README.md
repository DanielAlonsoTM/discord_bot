# Discord Bot

### Necessary permissions
* Scopes
    * bot
    * applications.commands

* Bot Permissions
    * General Permissions
        * Read Messages/View Channels
    * Text Permissions
        * Send Messages
        * Manage Messages
        * Embed Links
        * Attach Files
        * Read Message History
        * Mention Everyone
        * Use External Stickers
        * Add Reactions
    * Voice Permissions
        * Connect
        * Speak

### Python and pip version
* Python: 3.11.2
* pip3: 22.3.1

### Packages to install
* discord: 2.2.2
* jproperties: 2.1.1
* validators: 0.20.0
* youtube-dl: 2021.12.17
* PyNaCl: 1.5.0

Command to install: `pip3 install discord jproperties validators youtube-dl`
